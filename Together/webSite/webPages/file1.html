## 1- Definitions : **Page web** : est un document qui peut etre affiche par un navigateur, il est ecrit en HTML et peut inclure diverses autres ressources telles que des feuilles de styles(CSS), des scripts et du contenu multimedia. **Site web** : un ensemble de pages web reliees entre elles qui partagent un nom de domaine, chaque page fournit des liens explicites qui permettent a l'utilisateur de naviguer entre les pages web du site. Serveur web : il peut faire reference a des composants logiciels ou materiels ou les deux qui fonctionne ensemble. **Hypertext** : document which contents links to others documents, HTML is used to build a hypertext. **HTTP** : Hypertext Transfer Protocol, standard rules that define how Hypertext will be shared. **HTTP server** : web server that receives HTTP requests and can send resources that it has stored. **World Wide We** : information system that enables content sharing over the internet, making it easily accessible to the public. **RFC** : Requests For Comments, a numbered series of documents describing the technical aspects and specifications of the Internet, or of different computer hardware. **NGINX** : open source web server. **URL** : Uniform Resource Locator.
